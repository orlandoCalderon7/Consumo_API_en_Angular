# Angular API Rest

¡Bienvenido! Este es un proyecto de práctica desarrollado en **Angular** que consume la API oficial de Harry Potter para listar a los miembros del personal (staff) de Hogwarts, mostrando sus detalles y fotografías en tiempo real.

## Características
- **Consumo de API REST:** Uso de `HttpClient` para obtener datos asíncronos.
- **Inyección de Servicios:** Arquitectura limpia separando la lógica de datos de la interfaz.
- **Renderizado Dinámico:** Uso de directivas como `*ngFor` y `*ngIf`.
- **Interfaz Responsiva:** Diseño básico con imágenes circulares y estados de "Sin casa".

## Tecnologías Utilizadas
- **Angular 18+** (Framework principal)
- **TypeScript** (Lógica de programación)
- **HTML5 & CSS3** (Estructura y estilos personalizados)
- **HP-API** (Fuente de datos: `https://hp-api.onrender.com/`)
